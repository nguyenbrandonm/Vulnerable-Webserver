# Code of Conduct

## Purpose
This repository supports an **intentionally vulnerable application security and Active Directory lab** created for authorized training, learning, and portfolio demonstration purposes. All content is intended to be used **only in controlled, isolated environments** where the user has explicit permission to test.

## Authorized Use
By interacting with this repository, you agree to:
- Use the material only on systems you own or are explicitly authorized to test
- Deploy the lab in isolated or private environments
- Treat all demonstrated techniques as educational, not operational

## Expected Conduct
Participants and contributors are expected to:
- Act ethically and responsibly at all times
- Respect applicable laws, policies, and authorization boundaries
- Communicate professionally in issues, pull requests, and discussions
- Keep contributions focused on learning, documentation, and defensive understanding

## Prohibited Conduct
The following behavior is not permitted:
- Using this repository or its contents for unauthorized access to real systems or networks
- Submitting content intended for real-world exploitation outside a lab environment
- Encouraging misuse, abuse, or evasion of legal or organizational controls
- Harassment, abuse, or unprofessional behavior
- Sharing real credentials, secrets, tokens, or sensitive information

## Scope
This Code of Conduct applies to all interactions related to this repository, including code, documentation, issues, pull requests, and discussions.

## Enforcement
The repository maintainer may remove content or restrict participation that violates this Code of Conduct or conflicts with the intended educational purpose of the lab.
